# Slides de Metodologia Científica

Coleção de apresentações utilizadas em disciplina de metodologia científica e preparação de dados. Cada item traz o título do slide e um resumo rápido do foco abordado.

1. [Data Cleaning & Representation](daltoolbox-data-cleaning.pdf): fundamentos de qualidade de dados, remoção de ruídos e formas de representar atributos para modelos mais confiáveis.
2. [Data Discretization & Smoothing](daltoolbox-data-discretization-and-smoothing.pdf): técnicas para discretizar variáveis contínuas e suavizar séries preservando significado estatístico.
3. [Data Normalization in Machine Learning](daltoolbox-data-normalization.pdf): métodos de normalização para equilibrar escalas de atributos e melhorar o desempenho de algoritmos.
4. [Data Sampling Strategies for Machine Learning](daltoolbox-data-sampling.pdf): comparação entre amostragem aleatória e estratificada, com exemplos de divisão de dados no DAL Toolbox.
5. [Dimensionality Reduction & Curvature Analysis](daltoolbox-dimensionality-reduction-and-curvature-analysis.pdf): uso de PCA e análise de curvatura para escolher o número ideal de componentes.
6. [Introduction: Outlier Detection](daltoolbox-introduction-outlier-detection.pdf): conceitos de detecção de outliers e impactos desses valores em estatísticas e modelos preditivos.
7. [DAL Toolbox: Leveraging Experiment Lines for Data Analytics](daltoolbox.pdf): visão geral da biblioteca DAL Toolbox e de como linhas de experimentos estruturam pipelines reprodutíveis.
8. [Harbinger: A Unified Framework for Event Detection in Time Series](harbinger.pdf): plataforma unificada para detectar eventos em séries temporais e comparar diferentes abordagens.
9. [TSPred-IT: Time Series Prediction with Integrated Tuning](tspredit.pdf): framework para previsão de séries temporais com pré-processamento, aumento de dados e ajuste automático de modelos.
